Genea is an open-source family tree visualization and editing tool that runs entirely in the browser. It allows you to create, edit and visualize genealogical data using the GEDCOM standard format without requiring any server-side components.

## Key Features

- **Visual Tree Editor**:
  - Interactive drag-and-drop interface
  - Automatic tree layout with Graphviz
  - Support for extended family relationships
  - Real-time visualization updates
- **GEDCOM Support**:

  - Import/export GEDCOM 5.5.5 files
  - Extended support for same-sex relationships
  - Standard-compliant data format
  - Direct file manipulation

- **Person Management**:

  - Add/edit personal details
  - Define relationships
  - Link family members
  - Track important dates

- **Modern Architecture**:
  - Completely serverless
  - Works offline
  - Mobile-friendly interface
  - Browser-based storage

## Who Should Use Genea

Genea is ideal for:

- Genealogy researchers building family trees
- Users wanting local control of genealogy data
- Those needing a simple visual tree editor
- Anyone preferring serverless applications

## Installation Requirements

- Modern web browser
- HTTPS web server (for hosting)
- Minimal storage space
- No database required
