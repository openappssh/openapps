Gramps Web is an open-source genealogy web application that serves as a companion to the Gramps desktop application. It enables collaborative family tree building and research through a modern web interface accessible from any device.

## Key Features

- **Interactive Visualization**:

  - Family tree charts (ancestor, descendant, fan, hourglass)
  - Dynamic mapping tools with historical overlays
  - DNA and chromosome visualization
  - Interactive data exploration

- **Collaboration Tools**:

  - Real-time collaborative editing
  - Advanced privacy controls
  - User role management
  - Integrated research blog
  - Bi-directional sync with Gramps Desktop

- **Research Features**:

  - Advanced search with wildcards
  - DNA match analysis
  - AI-powered chat assistant
  - Place visualization
  - Source documentation
  - Media management

- **Technical Features**:
  - Modern web components architecture
  - Multi-language support (40+ languages)
  - Mobile-friendly design
  - GEDCOM compatibility
  - API access
  - Custom data export

## Who Should Use Gramps Web

Gramps Web is ideal for:

- Family historians wanting collaborative research tools
- Genealogy groups needing shared access
- Users wanting web access to Gramps data
- Organizations managing genealogical records
- Researchers requiring advanced visualization

## Installation Requirements

- Docker for container deployment
- Modern web browser
- 1GB RAM minimum
- Storage for genealogical data
