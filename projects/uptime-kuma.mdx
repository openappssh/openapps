Uptime Kuma is a comprehensive self-hosted monitoring solution that keeps track
of your websites, APIs, and services. With its intuitive interface and powerful
alerting system, you can ensure your services stay online and receive instant
notifications when issues occur.

This monitoring tool provides everything you need to maintain visibility into
your infrastructure health, offering detailed analytics, beautiful status pages,
and flexible notification options that rival commercial solutions.

## Key Features

### Comprehensive Monitoring Options

Monitor various types of services including HTTP/HTTPS websites, TCP ports,
ping, DNS records, and more. Set custom intervals, timeouts, and configure
detailed monitoring parameters for each service.

### Beautiful Status Pages

Create public or private status pages that your users can access to check
service availability. Customize the appearance and choose which monitors to
display publicly.

### Flexible Notification System

Get notified through 90+ notification channels including email, Discord, Slack,
Telegram, webhooks, and many more. Set up notification groups and customize
alerts for different types of incidents.

### Multi-User Support

Manage monitoring with multiple users, each with their own permissions and
access levels. Perfect for teams that need collaborative monitoring capabilities.

### Detailed Analytics & Reporting

View comprehensive uptime statistics, response time charts, and historical data.
Track trends over time and generate reports for stakeholders.

### Certificate Monitoring

Monitor SSL certificates and receive alerts before they expire. Ensure your
websites maintain secure connections without unexpected outages.

### Maintenance Windows

Schedule maintenance periods to prevent false alerts during planned downtime.
Set recurring schedules or one-time maintenance windows.

### Docker & Proxy Support

Easy deployment with Docker and Docker Compose. Supports reverse proxy setups
with automatic HTTPS through Let's Encrypt integration.

### Import/Export Configuration

Backup your monitoring configuration and easily migrate between instances.
Support for importing from other monitoring solutions.

### Mobile-Friendly Interface

Access your monitoring dashboard from any device with a responsive design that
works perfectly on mobile phones and tablets.

Uptime Kuma delivers enterprise-grade monitoring capabilities in a simple,
self-hosted package. Whether you're monitoring a few personal websites or
managing infrastructure for your organization, it provides the reliability
and features you need to stay informed about service health.
