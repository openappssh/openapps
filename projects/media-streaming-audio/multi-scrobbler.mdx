multi-scrobbler is a comprehensive music scrobbling solution that tracks listening activity from multiple sources and submits to multiple scrobbling services. Built with Node.js and Docker support, it provides centralized music tracking across various players and platforms.

## Key Features

- **Multiple Sources**: Support for various music players and streaming services
- **Multiple Destinations**: Scrobble to Last.fm, ListenBrainz, Maloja, and more
- **Real-Time Tracking**: Live scrobbling with duplicate detection
- **Docker Ready**: Easy deployment with Docker containers
- **Comprehensive Configuration**: Flexible setup for various sources and destinations
- **Monitoring**: Built-in monitoring and logging capabilities

## Technical Specifications

- **Language**: Node.js
- **License**: MIT
- **Deployment**: Docker, Node.js
- **Architecture**: Node.js application with API integrations
