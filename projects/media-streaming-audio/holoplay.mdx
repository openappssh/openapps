HoloPlay is a privacy-focused application that enables YouTube audio streaming through the Invidious API. Built as a Progressive Web App with Node.js, it provides a way to listen to YouTube content without directly accessing YouTube, offering enhanced privacy and a streamlined audio-focused experience.

## Key Features

- **Privacy-Focused Streaming**: Uses Invidious API as a proxy to access YouTube content without direct YouTube tracking
- **Audio-Only Experience**: Streamlined interface focused on audio playback from YouTube sources
- **Progressive Web App**: Modern PWA technology for app-like experience across devices
- **YouTube Integration**: Access to YouTube's vast audio content library through privacy-respecting proxy
- **Lightweight Design**: Minimal, efficient application focused on essential audio streaming features
- **Mobile Optimized**: Responsive design optimized for mobile and desktop use

## Technical Specifications

- **Language**: Node.js
- **License**: MIT
- **Deployment**: Docker, Node.js
- **Architecture**: Progressive Web App with Invidious API integration

## Use Cases

- **Privacy-Conscious Users**: YouTube audio without direct YouTube tracking
- **Audio-Only Listening**: Focus on audio content from YouTube
- **Mobile Audio**: Lightweight YouTube audio streaming on mobile devices
- **Background Playback**: Audio streaming without video overhead
- **Alternative Access**: YouTube content access through privacy-respecting proxy

## Unique Advantages

- **Privacy Protection**: Invidious proxy for enhanced privacy
- **Audio Focus**: Streamlined audio-only YouTube experience
- **PWA Technology**: Modern web app capabilities
- **Lightweight**: Minimal resource usage and clean interface
- **Open Source**: Transparent and customizable

⚠️ **Note**: This application depends on external Invidious instances and YouTube content availability.
