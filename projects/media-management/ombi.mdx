Ombi is a mature and feature-rich content request system designed to bridge the gap between media server users and content automation tools. It provides an elegant interface for requesting movies, TV shows, and music while seamlessly integrating with popular media automation platforms to fulfill those requests automatically.

## Key Features

- **Comprehensive Media Server Integration**:

  - Full Plex server integration and library access
  - Native Emby support and synchronization
  - Jellyfin compatibility and management
  - Multi-server support for complex environments
  - Real-time library synchronization and updates
  - Cross-platform media server management

- **Advanced Request Management System**:

  - Intuitive movie and TV show request interface
  - Music request support for audio libraries
  - Intelligent request queue and prioritization
  - Real-time status tracking and updates
  - Comprehensive request history and analytics
  - Bulk request processing and management

- **Extensive Automation Tool Integration**:

  - Native Sonarr integration for TV show automation
  - Radarr support for movie acquisition
  - SickRage connection for legacy TV management
  - CouchPotato support for movie downloading
  - Lidarr integration for music automation
  - Custom automation workflow support

- **Sophisticated User Management**:

  - Comprehensive user authentication and authorization
  - Granular permission system and access control
  - Configurable request limits and quotas
  - Detailed user profiles and preferences
  - Social authentication integration (Google, Discord, etc.)
  - Multi-user collaboration and sharing

- **Comprehensive Notification System**:

  - Email notification support and templates
  - Discord integration for community servers
  - Slack support for team environments
  - Telegram notifications for mobile alerts
  - Push notification support for mobile apps
  - Customizable notification preferences and timing

- **Modern User Interface and Experience**:

  - Responsive web interface for all devices
  - Dedicated mobile applications for iOS and Android
  - Advanced search and content discovery
  - Popular and trending content recommendations
  - Visual calendar view of releases and requests
  - Customizable themes and personalization

- **Self-Hosting Benefits**:
  - Complete control over request workflows and user management
  - Privacy protection for viewing preferences and user data
  - Custom configuration for family and community usage
  - Integration with existing media automation infrastructure
  - No dependency on external request services
  - Enhanced security for private media libraries and communities

## Technical Specifications

- **License**: GPL-2.0
- **Deployment**: Docker containers, native binaries
- **Platforms**: Cross-platform (.NET Core)
- **Requirements**: .NET runtime, database
- **Database**: SQLite, MySQL, PostgreSQL
- **Architecture**: C# backend with Angular frontend

## Use Cases

- **Family Media Centers**: Centralized media requests for household members
- **Community Servers**: Shared media libraries with user request management
- **Friend Groups**: Collaborative content discovery and requesting
- **Media Automation**: Integration with existing download and organization tools
- **Content Discovery**: Finding and requesting new movies, shows, and music
- **Library Management**: Organized content acquisition and user interaction

- **Mature Platform**: Well-established with extensive feature set and community
- **Multi-Media Support**: Movies, TV shows, and music request capabilities
- **Extensive Integration**: Wide range of automation tool connections
- **Mobile Apps**: Dedicated mobile applications for enhanced accessibility
