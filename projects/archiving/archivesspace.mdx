# ArchivesSpace

ArchivesSpace is an open source archives information management system designed for managing and providing web access to archives, manuscripts and digital objects. Built by archivists for archives, it helps institutions better manage and provide access to their collections.

## Key Features

- Collection management for archives and manuscripts
- Digital object management
- Web-based public access interface
- Standards-compliant metadata (DACS, ISAD(G), EAD, MARC)
- Customizable workflows
- API access
- Import/export capabilities

## Getting Started

ArchivesSpace is free to download and use, though it has a membership model to support ongoing development and maintenance. To get started:

1. Download the latest release from [GitHub](https://github.com/archivesspace/archivesspace)
2. Follow the [installation instructions](https://archivesspace.github.io/tech-docs/administration/getting_started.html)
3. Configure the application for your institution
4. Begin managing your collections!
