Zot is a vendor-neutral OCI-native container image registry that strictly adheres to the OCI Distribution Specification. It provides a scalable platform for storing and distributing container images and artifacts in OCI format.

## Key Features

- **OCI Compliance**:

  - Full OCI Distribution Specification support
  - Native OCI image format storage
  - Vendor-neutral implementation
  - Standards-based distribution

- **Registry Capabilities**:

  - Scale-out architecture
  - Multi-platform support
  - Image signing and verification
  - Access control management
  - Security scanning
  - High availability

- **Integration Support**:

  - Kubernetes compatibility
  - Helm chart repository
  - Docker registry compatibility
  - CI/CD pipeline integration
  - REST API access
  - Command line tools
