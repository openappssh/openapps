Keila is an open source newsletter management system that provides an
alternative to services like Mailchimp and Sendinblue. It offers comprehensive
features for managing subscribers, creating campaigns, and sending newsletters
through various delivery providers.

## Key Features

- **Campaign Management**:

  - WYSIWYG email editor
  - Custom HTML templates
  - Subscriber segmentation
  - Campaign scheduling
  - Analytics and tracking
  - A/B testing

- **Subscriber Management**:

  - Custom signup forms
  - List management
  - Contact grouping
  - Import/export tools
  - Double opt-in
  - Unsubscribe handling

- **Email Delivery**:

  - Multiple delivery providers
  - SMTP support
  - AWS SES integration
  - Sendgrid support
  - Mailgun integration
  - Postmark support

- **Technical Features**:
  - REST API access
  - Docker deployment
  - Database-driven
  - Mobile-friendly interface
  - Multi-language support
  - Extensible architecture

## Who Should Use Keila

Keila is ideal for:

- Organizations needing newsletter tools
- Businesses wanting email control
- Privacy-focused newsletter senders
- Self-hosted solution seekers
- Small to medium newsletters
- Content creators and bloggers

## Installation Requirements

- Docker and Docker Compose
- PostgreSQL database
- Modern web browser
- 512MB RAM minimum
- Storage for subscriber data
