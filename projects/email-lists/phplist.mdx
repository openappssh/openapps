phpList is a powerful open source email marketing and newsletter manager that helps organizations create, send, and analyze email campaigns. With over 25 billion campaign messages sent in 2018, it's a proven solution used globally.

## Key Features

- **Campaign Management**:

  - Web-based and CLI interfaces
  - Template system
  - Campaign scheduling
  - Content personalization
  - A/B testing
  - RSS feed integration

- **Subscriber Management**:
  - List segmentation
  - Custom attributes
  - Import/export via CSV/Excel
  - Double opt-in support
  - Preference management
- **Analytics & Processing**:

  - Real-time tracking
  - Subscriber behavior analysis
  - Automated bounce handling
  - Policy-based processing
  - Delivery reporting

- **System Features**:
  - Amazon SES integration
  - Multi-language support
  - Plugin architecture
  - Multiple APIs
  - Domain-based throttling
  - Attachment support

## Who Should Use phpList

phpList is ideal for:

- Organizations needing self-hosted or hosted options
- Teams requiring advanced subscriber management
- Marketers wanting detailed analytics
- Users needing multi-language support
- Cost-conscious newsletter operators

## Installation Requirements

- PHP
- MySQL database
- Web server (Apache/Nginx)
- SMTP server access

The system can be deployed either self-hosted or through phpList Hosted for a managed solution.
