Mylar3 is an open source comic book automation tool that helps manage and download your digital comic collection. It provides comprehensive monitoring of series, story arcs, and new releases while integrating with various download clients and metadata tools.

## Key Features

- **Comic Management**:

  - Series watchlist monitoring
  - Weekly pull-list tracking
  - Story arc support
  - Library scanning
  - Missing issue detection
  - Automated renaming

- **Download Integration**:

  - SABnzbd support
  - NZBGet integration
  - Torrent client support
  - Multiple newznab indexers
  - Failed download handling
  - Direct download options

- **Metadata Features**:

  - ComicTagger integration
  - Automatic tagging
  - Series info generation
  - Custom metadata rules
  - Format standardization
  - Library organization

- **Additional Capabilities**:
  - Multi-platform support
  - Notification system
  - Web interface
  - API access
  - TPB/GN support
  - Custom file naming

## Who Should Use Mylar3

Mylar3 is ideal for:

- **Comic Collectors** managing digital libraries
- **Automation Enthusiasts** wanting organized collections
- **Series Followers** tracking new releases
- **Digital Archivists** maintaining metadata
- **Comic Readers** automating downloads

## Getting Started

The platform can be deployed via Docker or directly on a Python environment. It integrates with various download clients and provides comprehensive documentation for setup and configuration.

Whether you're managing a small collection or large library, Mylar3 provides the automation tools needed for maintaining an organized digital comic collection.
