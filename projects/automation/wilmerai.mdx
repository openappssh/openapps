WilmerAI is an open source platform that enables prompts to be routed to specialized workflows and allows multiple LLMs to work together to generate responses. It provides a flexible workflow system with support for parallel processing, memory management, and integration with external tools.

## Key Features

- **Workflow Management**:

  - Visual flow builder
  - Custom Python scripts
  - Parallel processing
  - Memory management
  - Chat summarization
  - Workflow locks

- **LLM Integration**:

  - Multi-model support
  - OpenAI compatibility
  - Ollama compatibility
  - KoboldCpp support
  - Preset management
  - Context handling

- **Development Features**:

  - Python extensibility
  - Custom workflows
  - Conditional routing
  - Image processing
  - API integration
  - Memory tools

- **Security Features**:
  - Workflow locks
  - Memory encryption
  - API authentication
  - Secure endpoints
  - Access control
  - Audit logging

## Who Should Use WilmerAI

WilmerAI is ideal for:

- **Developers** building LLM applications
- **Teams** needing workflow automation
- **Organizations** requiring LLM routing
- **Researchers** experimenting with models
- **Enthusiasts** exploring AI capabilities

## Getting Started

The platform can be quickly deployed using Python with minimal configuration required. It provides flexible workflow configuration through JSON files and comprehensive documentation for setup and usage.

Whether you're building simple automations or complex multi-model workflows, WilmerAI provides the infrastructure needed for sophisticated LLM orchestration while maintaining flexibility through its extensible architecture.
