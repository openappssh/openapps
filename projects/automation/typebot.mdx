Typebot is a powerful open-source platform for building conversational apps and forms. It provides a visual builder that makes it easy to create engaging chatbots without coding, perfect for surveys, lead generation, and customer support.

## Key Features

- **Visual Builder**:

  - Intuitive drag-and-drop interface
  - Real-time preview
  - Logic flow visualization
  - Variable management
  - Conditional branching
  - Pre-built templates

- **Rich Input Types**:

  - Text and email inputs
  - Multiple choice questions
  - File and image uploads
  - Date and time pickers
  - Rating and NPS scales
  - Payment collection

- **Advanced Logic**:

  - Conditional paths
  - Variable calculations
  - Jump to specific blocks
  - Hidden fields
  - URL parameters
  - Custom scripts

- **Flexible Deployment**:
  - Embeddable chat widget
  - Full-page experiences
  - Custom domain support
  - White-label options
  - Analytics dashboard
  - GDPR compliant
