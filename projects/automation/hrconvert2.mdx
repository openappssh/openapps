HRConvert2 is an open source file conversion server that supports hundreds of file formats through a simple drag-and-drop interface. It provides local file conversion capabilities with additional features like OCR, virus scanning, and temporary file sharing.

## Key Features

- **File Conversion**:

  - 445+ supported formats
  - Local processing
  - Drag-and-drop interface
  - Batch conversion
  - Format-specific options
  - OCR capabilities

- **Security Features**:

  - ClamAV integration
  - On-demand scanning
  - No external connections
  - No tracking/analytics
  - Private scratch space
  - Temporary file sharing

- **Interface Options**:

  - 13 language options
  - 4 color schemes
  - Minimal UI design
  - API access
  - URL language switching
  - Custom branding

- **Format Support**:
  - Audio/video formats
  - Document types
  - Spreadsheets
  - Presentations
  - Archives
  - Images
  - 3D models
  - Subtitles

## Who Should Use HRConvert2

HRConvert2 is ideal for:

- **Organizations** needing private conversion tools
- **Developers** requiring format conversion APIs
- **Teams** wanting self-hosted solutions
- **Users** concerned with privacy
- **Admins** managing file processing

## Getting Started

The platform can be installed on any PHP server with the necessary conversion tools configured. It operates without databases or external dependencies, making deployment straightforward for those familiar with PHP environments.

Whether you're converting documents, processing media files, or need secure file sharing capabilities, HRConvert2 provides a comprehensive self-hosted solution for file conversion needs.
