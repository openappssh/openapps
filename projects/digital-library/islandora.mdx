Islandora is a comprehensive digital asset management solution that combines the content management power of Drupal with the digital preservation capabilities of Fedora Repository. It's designed for libraries, archives, and museums to build sophisticated digital collections.

## Key Features

- **Versatile Content Support**:

  - Large image handling with zooming
  - Book and manuscript viewers
  - Audio/video streaming
  - PDF display and download
  - Newspaper browsing
  - 3D object support

- **Preservation-Ready**:

  - PREMIS metadata support
  - Technical metadata extraction
  - Preservation policy framework
  - Migration pathways
  - Fixity checking
  - Archival storage

- **Rich Discovery**:

  - Apache Solr integration
  - Faceted search interface
  - Hierarchical collections
  - Compound objects
  - IIIF image API
  - Metadata crosswalks

- **Flexible Architecture**:
  - Drupal 8/9 integration
  - Microservices approach
  - RESTful APIs
  - Linked data support
  - Modular design
  - Multi-site capable
