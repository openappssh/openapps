SHKeeper is an open-source cryptocurrency payment processor that enables businesses to accept direct crypto payments without intermediaries or transaction fees. It supports multiple cryptocurrencies including BTC, ETH, LTC, DOGE, XMR, XRP, and various stablecoins.

## Key Features

- **Payment Processing**:

  - Multi-currency support
  - Direct crypto payments
  - Partial payments
  - Auto-payments
  - Custom exchange rates
  - Fee management

- **Integration Options**:

  - WHMCS module
  - WooCommerce plugin
  - OpenCart module
  - PrestaShop module
  - REST API
  - Callback system

- **Technical Capabilities**:

  - Docker deployment
  - Kubernetes support
  - Wallet encryption
  - API access
  - Multipayout support
  - Performance monitoring

- **Security Features**:
  - Non-custodial
  - Self-hosted
  - Wallet encryption
  - No KYC required
  - Access control
  - API authentication
