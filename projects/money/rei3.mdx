REI3 is an open-source low-code platform that enables organizations to build and host powerful business applications. It provides comprehensive tools for creating multi-user applications with features like workflows, access control, and end-to-end encryption while maintaining full control over data and deployment.

## Key Features

- **Application Development**:

  - Low-code builder
  - Workflow engine
  - Custom forms
  - Business rules
  - Data visualization
  - Mobile optimization

- **Enterprise Features**:

  - End-to-end encryption
  - Access control
  - Role management
  - LDAP integration
  - Custom branding
  - Multi-user support

- **Technical Capabilities**:

  - REST API
  - Docker deployment
  - Clustering support
  - PostgreSQL database
  - Websocket communication
  - Performance optimization

- **Security & Privacy**:
  - MFA support
  - Password policies
  - Brute-force protection
  - Data encryption
  - Access logging
  - Role-based access
