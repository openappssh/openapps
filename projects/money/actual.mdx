Actual is an open-source personal finance tool built on local-first principles, providing powerful envelope budgeting capabilities while ensuring complete data privacy and ownership. It combines modern technology with traditional budgeting methods to help users manage their finances effectively.

## Key Features

- **Budgeting Tools**:

  - Envelope budgeting system
  - Account management
  - Transaction tracking
  - Budget planning
  - Financial reporting
  - Goal setting

- **Technical Architecture**:

  - Local-first design
  - Cross-device synchronization
  - Data export capabilities
  - Self-hosting options
  - Desktop applications
  - Offline support

- **Privacy & Security**:

  - Complete data ownership
  - Private by design
  - No usage tracking
