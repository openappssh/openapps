Galette is an open-source membership management system designed for non-profit organizations. It provides comprehensive tools for tracking members, managing dues, and handling organizational tasks while remaining free and customizable.

## Key Features

- **Membership Management**:

  - Member information tracking
  - Dues and contribution management
  - Customizable member cards
  - Group organization
  - Dynamic custom fields
  - Member status tracking

- **Administrative Tools**:

  - Member directory
  - Contribution history
  - Access control system
  - Mailing list integration
  - PDF document generation
  - Email communication

- **Technical Capabilities**:

  - Multi-language support
  - Multiple database support
  - Data export options
  - PDF generation
  - Email integration
  - Customizable templates

- **Organization Features**:
  - Group management
  - Member tags
  - Dynamic fields
  - Status tracking
  - Contribution types
  - Search capabilities
