Ghostfolio is a modern wealth management platform designed to help users track and analyze their investment portfolios. It provides comprehensive tools for monitoring stocks, ETFs, and cryptocurrencies while offering insights for data-driven investment decisions.

## Key Features

- **Portfolio Management**:

  - Multi-account tracking
  - Performance analytics (ROAI)
  - Transaction management
  - Portfolio risk analysis
  - Investment insights
  - Multi-currency support

- **Technical Capabilities**:

  - Progressive Web App (PWA)
  - Mobile-first design
  - Data import/export
  - Public API access
  - Docker deployment
  - PostgreSQL database

- **User Experience**:

  - Dark mode support
  - Zen mode
  - Customizable views
  - Various charts
  - Static analysis
  - Mobile responsiveness

- **Security & Privacy**:
  - Data ownership
  - Self-hosting option
  - Access controls
  - Privacy focused
  - Regular updates
  - Data export capabilities
