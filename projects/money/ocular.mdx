Ocular is a modern, open-source budgeting application that helps users track and visualize their finances across multiple years. It offers a clean, intuitive interface with powerful features like Sankey diagrams, detailed analytics, and comprehensive budget planning tools.

## Key Features

- **Budget Management**:

  - Multi-year tracking
  - Budget planning
  - Expense analysis
  - Sankey diagrams
  - Privacy mode
  - Detailed analytics

- **User Experience**:

  - Clean interface
  - Dark/light theme
  - Mobile support
  - PWA installation
  - Multi-language
  - Multi-currency

- **Technical Capabilities**:

  - Docker deployment
  - Data import/export
  - Google Sheets import
  - User authentication
  - JSON data export
  - Fast performance

- **Privacy & Security**:
  - Self-hosted
  - User management
  - Privacy mode
  - Local data storage
  - Export backups
  - Access control
