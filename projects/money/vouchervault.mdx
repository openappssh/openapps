VoucherVault is an open-source Django application that helps users digitally organize and manage their vouchers, coupons, loyalty cards and gift cards. It features a mobile-optimized interface with support for file uploads, QR codes, transaction tracking and expiry notifications.

## Key Features

- **Voucher Management**:

  - Digital voucher storage
  - Gift card tracking
  - Loyalty card support
  - Transaction history
  - File attachments
  - Item sharing

- **Code Features**:

  - QR code generation
  - Barcode support
  - Code scanning
  - Type detection
  - Mobile scanning
  - Digital display

- **Technical Capabilities**:

  - Docker deployment
  - REST API access
  - OIDC integration
  - Multi-language support
  - Dark/light themes
  - PostgreSQL/SQLite

- **User Experience**:
  - Mobile optimization
  - File uploads
  - Expiry notifications
  - Transaction tracking
  - Item sharing
  - Intuitive interface
