I Hate Money is a minimalist web application designed to simplify shared budget management. It focuses on helping groups track shared expenses, calculate balances, and settle bills while maintaining a simple and straightforward user experience.

## Key Features

- **Expense Management**:

  - Simple expense tracking
  - Automatic balance calculation
  - Bill settlement assistance
  - Multi-currency support
  - Project-based organization
  - Quick expense entry

- **Group Collaboration**:

  - No account required for participants
  - Email-based invitations
  - Multiple projects support
  - Balance reporting
  - Expense history
  - Shared access control

- **Technical Capabilities**:

  - Multiple database support
  - Docker deployment
  - Simple web interface
  - Mobile responsiveness
  - Email notifications
  - Data export options

- **User Experience**:
  - Minimal interface
  - Quick data entry
  - Easy navigation
  - Clear balance views
  - Straightforward setup
  - Instant calculations
