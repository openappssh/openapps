DePay is an open-source Web3 payment protocol that enables direct peer-to-peer cryptocurrency payments with automatic token conversion. It allows merchants to accept any cryptocurrency while receiving their preferred token through decentralized exchange integration.

## Key Features

- **Payment Processing**:

  - Multi-chain payment support
  - Automatic token conversion
  - Direct peer-to-peer transactions
  - Widget-based integration
  - Payment routing optimization
  - Real-time conversion rates

- **Technical Infrastructure**:

  - React-based widgets
  - Web3 wallet integration
  - API support
  - Self-hosting capability
  - Multiple wallet compatibility
  - Cross-chain functionality

- **Security & Privacy**:

  - Non-custodial architecture
  - Smart contract based operations
  - Trustless design
  - Open source transparency
  - Permissionless system

- **Integration Tools**:
  - Payment widgets
  - Sale widgets
  - Login widgets
  - Connect widgets
  - Select widgets
  - Customizable styling
