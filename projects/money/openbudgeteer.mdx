OpenBudgeteer is a modern, open-source budgeting application that implements the Bucket Budgeting Principle, similar to YNAB's methodology. Built with .NET Core and Blazor, it offers a robust self-hosted solution for personal finance management.

## Key Features

- **Budget Management**:

  - Bucket-based budgeting
  - Transaction tracking
  - Account management
  - Budget planning
  - Custom tags
  - Multi-currency support

- **Financial Tools**:

  - Financial reporting
  - Data analytics
  - Transaction import
  - Account tracking
  - Budget visualization
  - Balance monitoring

- **Technical Capabilities**:

  - Docker deployment
  - Database options (MariaDB/MySQL/PostgreSQL)
  - REST API access
  - Theme customization
  - MVVM architecture
  - Blazor Server frontend

- **User Experience**:
  - Clean interface
  - Custom theming
  - Transaction management
  - Account overview
  - Report generation
  - Budget tracking
