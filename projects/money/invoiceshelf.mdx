InvoiceShelf is a comprehensive open-source invoicing and expense management platform forked from Crater. It focuses on stability and new features while helping businesses create professional invoices, track expenses, and manage payments efficiently.

## Key Features

- **Invoice Management**:

  - Professional invoice creation
  - Estimate generation
  - Custom invoice templates
  - Email integration
  - Payment tracking
  - Tax management

- **Business Tools**:

  - Expense tracking
  - Customer management
  - Multiple companies
  - Custom fields
  - Financial reporting
  - Address customization

- **Technical Capabilities**:

  - Mobile applications
  - REST API access
  - Multi-language support
  - Docker deployment
  - Template customization
  - Automatic updates

- **User Experience**:
  - Installation wizard
  - Customer portal
  - Email configuration
  - Default notes
  - Custom fields
  - Performance optimization
