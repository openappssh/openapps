ExpenseOwl is an open-source expense tracking system focused on simplicity and essential features. It provides monthly pie-chart visualizations and cashflow tracking without the complexity of full-featured budgeting tools.

## Key Features

- **Expense Tracking**:

  - Simple expense entry with essential details
  - Category-based pie chart visualization
  - Monthly cashflow indicators
  - Customizable tags
  - Currency symbol customization
  - Flexible start date configuration

- **Technical Infrastructure**:

  - Self-hosted solution
  - Docker container support
  - REST API integration
  - Progressive Web App (PWA)
  - Multi-architecture support
  - Flat file storage system

- **Visualization & Interface**:

  - Interactive pie charts
  - Category filtering
  - Dark/light theme support
  - Mobile responsive design
  - Table view for detailed listing
  - Month-by-month navigation

- **Data Management**:
  - CSV/JSON data export
  - Data import capabilities
  - Single-user focused
  - UUID-based tracking
  - Persistent storage
  - Browser search support
