Receipt Wrangler is an open-source receipt management and expense splitting application that helps users digitize, organize, and share receipts. It features advanced capabilities like OCR scanning, AI-assisted data extraction, and automated email processing.

## Key Features

- **Receipt Management**:

  - OCR receipt scanning
  - AI data extraction
  - Email integration
  - Receipt splitting
  - Multi-user support
  - Group sharing

- **Processing Capabilities**:

  - Automated scanning
  - Email processing
  - Data extraction
  - Category management
  - Transaction tracking
  - Expense analytics

- **Technical Features**:

  - Docker deployment
  - API access
  - Group management
  - Email integration
  - OCR engine
  - Multi-user architecture

- **User Experience**:
  - Receipt upload
  - Expense tracking
  - Group sharing
  - Transaction history
  - Custom categories
  - Splitting tools
