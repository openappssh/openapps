WYGIWYH (What You Get Is What You Have) is an opinionated finance tracker that takes a no-budget approach to money management. It helps users track their finances across multiple currencies while maintaining a straightforward philosophy: use what you earn this month for this month.

## Key Features

- **Financial Management**:

  - Multi-currency support
  - Custom transaction rules
  - DCA investment tracking
  - Account management
  - Transaction categorization
  - Exchange rate handling

- **Technical Capabilities**:

  - Docker deployment
  - REST API access
  - Mobile optimization
  - Multi-language support
  - Automated tasks
  - Database backups

- **Transaction Features**:

  - Custom currencies
  - Automated adjustments
  - Import/export support
  - Transaction history
  - Rule-based processing
  - Bulk operations

- **User Experience**:
  - Clean interface
  - Financial reporting
  - Account overview
  - Investment tracking
  - Currency conversion
  - Data visualization
