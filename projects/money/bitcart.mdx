Bitcart is an open-source cryptocurrency payment processor that enables merchants to accept various cryptocurrencies while maintaining full control over their funds. It provides a comprehensive platform for payment processing, invoice management, and store integration.

## Key Features

- **Payment Processing**:

  - Support for multiple cryptocurrencies (BTC, LTC, BCH, XMR, ETH, TRX, USDT)
  - Automated payment detection
  - Invoice generation and management
  - Custom checkout flow automation
  - Real-time payment tracking

- **Technical Infrastructure**:

  - Docker-based deployment
  - REST API integration
  - Self-hosted solution
  - Multi-user support
  - Role-based access control

- **Security & Privacy**:

  - Self-custody of funds
  - No third-party dependencies
  - Open-source transparency
  - Secure key storage
  - Access control management

- **Merchant Tools**:
  - Store integration
  - Admin dashboard
  - Payment statistics
  - Multiple store support
  - Customizable checkout experience
