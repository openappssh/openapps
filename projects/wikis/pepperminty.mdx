Pepperminty Wiki is a unique wiki engine that packs complete functionality into a
single PHP file. Inspired by Minty Wiki, it offers a lightweight yet feature-rich
solution for knowledge management while maintaining simplicity in deployment and
maintenance.

## Key Features

- **Single-File Architecture**:

  - Complete wiki engine in one PHP file
  - Simple installation and backup
  - Modular design for extensibility
  - Built-in configuration system
  - Minimal server requirements

- **Content Management**:

  - File upload capabilities
  - Page revision history
  - Page tagging system
  - Dynamic help pages
  - Markdown support
  - Search functionality

- **Collaboration Tools**:

  - Multi-user support
  - User watchlists
  - Page permissions
  - File sharing
  - Change tracking

- **Advanced Capabilities**:
  - Theme customization
  - Mobile-friendly interface
  - API access
  - Auto-update system
  - Docker deployment support
  - RSS feeds for changes

## Who Should Use Pepperminty Wiki

Pepperminty Wiki is ideal for:

- **Small Teams** needing a lightweight documentation solution
- **Personal Users** wanting a simple self-hosted wiki
- **Projects** requiring easy-to-maintain documentation
- **Organizations** seeking a minimal yet capable wiki platform

## Getting Started

Pepperminty Wiki offers multiple deployment options:

1. **Traditional Installation**:

   - Download single PHP file
   - Place in web-accessible directory
   - Configure web server
   - Start creating content

2. **Docker Deployment**:
   - Pull official Docker image
   - Configure environment variables
   - Run container
   - Access via web browser

The system requires minimal setup and maintenance while providing robust wiki
functionality. Its single-file nature makes it particularly easy to backup,
move, or upgrade as needed.
