Litlyx is a modern web analytics platform focused on providing a developer-friendly,
cookie-free solution for tracking website metrics. It offers both cloud-hosted and
self-hosted options with quick setup and comprehensive analytics capabilities.

## Key Features

- **Easy Integration**:

  - 30-second setup process
  - Universal script tag installation
  - NPM package support
  - Framework-agnostic implementation
  - WordPress compatibility
  - Custom event tracking

- **Core Analytics**:

  - Page visit tracking
  - Real-time user monitoring
  - Unique visitor counting
  - Bounce rate analysis
  - Session duration metrics
  - Geographic data

- **Advanced Capabilities**:

  - Custom event tracking
  - Metadata support
  - API access
  - Multiple deployment options
  - Framework integrations
  - Real-time dashboard

- **Privacy Features**:
  - Cookie-free tracking
  - GDPR compliance
  - Self-hosting option
  - Data ownership
  - Configurable tracking
  - Privacy-first design

## Who Should Use Litlyx

Litlyx is ideal for:

- **Developers** seeking simple analytics integration
- **Organizations** requiring privacy-focused tracking
- **Website Owners** wanting comprehensive metrics
- **Teams** needing real-time analytics

## Getting Started

Litlyx offers multiple implementation options:

1. **Script Tag Integration**:

   ```html
   <script
     defer
     data-project="your_project_id"
     src="https://cdn.jsdelivr.net/gh/litlyx/litlyx-js/browser/litlyx.js"
   ></script>
   ```

2. **NPM Installation**:

   ```bash
   npm i litlyx-js
   ```

3. **Self-Hosted Docker**:
   ```bash
   docker-compose up
   ```

The platform provides immediate analytics capabilities while maintaining
flexibility for both cloud and self-hosted deployments.
