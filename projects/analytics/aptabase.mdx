Aptabase is an open-source analytics platform specifically designed for Mobile,
Desktop and Web applications. It provides a privacy-first alternative to services
like Firebase Analytics and Google Analytics, focusing on collecting only essential
usage data without compromising user privacy.

## Key Features

- **Privacy-First Analytics**:

  - No personal data collection
  - GDPR, CCPA, and PECR compliant
  - Session-based tracking
  - No unique identifiers
  - Minimal data collection

- **Extensive SDK Support**:

  - Swift (iOS/macOS)
  - Android (Kotlin)
  - React Native
  - Flutter
  - Tauri
  - NativeScript
  - .NET MAUI
  - Electron
  - Web Apps
  - Unreal Engine
  - Unity Engine

- **Simple Dashboard**:

  - Essential metrics visualization
  - Built-in reporting
  - User-friendly interface
  - Real-time analytics
  - Custom properties support

- **Developer-Focused**:
  - Open source codebase
  - MIT licensed SDKs
  - Easy integration
  - Self-hosted option
  - API access

## Who Should Use Aptabase

Aptabase is ideal for:

- **App Developers** needing privacy-compliant analytics
- **Organizations** seeking Firebase Analytics alternatives
- **Teams** wanting simple, focused app metrics
- **Projects** requiring open-source analytics solutions

## Getting Started

Aptabase offers two deployment options:

1. **Cloud Service**:

   - Managed by Aptabase team
   - Instant setup
   - Automatic updates
   - Free tier available

2. **Self-Hosted**:
   ```bash
   docker-compose up -d
   ```

The platform provides immediate functionality with minimal configuration while
maintaining strong privacy standards and comprehensive analytics capabilities.
