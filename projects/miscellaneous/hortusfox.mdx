HortusFox is a comprehensive plant management system designed for collaborative environments. It helps users track and manage plants across multiple locations while providing tools for task management, inventory control, and team communication.

## Key Features

- **Plant Management**:

  - Detailed plant profiles
  - Custom location tracking
  - Task management system
  - Inventory tracking
  - Calendar integration
  - History logging

- **Collaboration Tools**:

  - Multi-user support
  - Group chat functionality
  - Task assignments
  - Shared calendars
  - Activity tracking
  - Automated reminders

- **Additional Capabilities**:

  - Weather monitoring
  - Plant identification
  - Automated backups
  - REST API access
  - Customizable themes
  - Administrative controls
