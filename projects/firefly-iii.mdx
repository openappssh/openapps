Firefly III is a self-hosted financial manager that helps you track your income,
expenses, and overall financial health. Built with privacy in mind, this
open-source tool gives you complete control over your financial data while
offering powerful features for budgeting, tracking, and financial planning.

## Key Features

### Financial Management

- Double-entry bookkeeping system
- Multiple account types (asset, expense, revenue, liabilities)
- Split transactions for detailed tracking
- Recurring transactions
- Import data from banks via CSV or API
- Tags for transaction organization
- Customizable categories and transaction types
- Account reconciliation

### Budgeting Tools

- Monthly or custom period budgeting
- Budget tracking and reporting
- Auto-budget feature
- Set financial goals
- Create savings targets with progress tracking
- Bill management and reminders
- Future financial forecasting

### Reports & Insights

- Comprehensive spending and income reports
- Net worth calculations and tracking
- Category spending breakdown
- Tag-based reports
- Cash flow analysis
- Custom date range reporting
- Expense tracking by payee
- Export data to CSV

### Security & Integration

- Self-hosted for maximum privacy
- No data shared with third parties
- API for integrations with other tools
- Mobile-friendly interface
- 2FA authentication support
- Multiple currency support
- Data import/export capabilities
- Regular updates and security patches

Firefly III helps you gain insights into your spending patterns, set realistic
budgets, and work toward financial goals. With its focus on privacy and
self-hosting, you can manage your finances without sharing sensitive financial
information with commercial services. The clean interface makes it easy to get
started while still offering advanced features for power users.
