Litecart is an ultra-lightweight shopping cart solution that proves e-commerce doesn't need to be complicated. Built in Go as a single file application, it supports both traditional and cryptocurrency payments while maintaining extreme simplicity.

## Key Features

- **Minimalist Design**:

  - Single file application
  - No database required
  - Minimal configuration
  - Low resource footprint
  - Fast deployment
  - Easy maintenance

- **Payment Flexibility**:

  - Credit card processing
  - Cryptocurrency support
  - Multiple payment gateways
  - Secure transactions
  - Currency conversion
  - Payment notifications

- **Essential E-commerce**:

  - Product management
  - Shopping cart functionality
  - Streamlined checkout
  - Order processing
  - Email confirmations
  - Basic inventory tracking

- **Developer-Friendly**:
  - Written in Go
  - Docker containerized
  - Cross-platform binary
  - Simple API
  - Easy customization
  - Clear documentation
