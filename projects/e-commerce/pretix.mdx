Pretix is a comprehensive ticketing software designed for event organizers who value data privacy, flexibility, and control. It handles everything from ticket sales to check-in, with powerful features for events of all sizes.

## Key Features

- **Flexible Ticketing**:

  - Multiple ticket categories
  - Early bird pricing
  - Seating chart support
  - Add-on products
  - Voucher codes
  - Waiting list management

- **Multi-Channel Sales**:

  - Customizable web shop
  - Box office mode
  - Reseller management
  - Embeddable widgets
  - REST API
  - White-label options

- **Complete Event Tools**:

  - Mobile check-in apps
  - Badge generation
  - Real-time statistics
  - Attendee communication
  - Access control
  - Session management

- **Privacy & Compliance**:
  - GDPR compliant
  - Data minimization
  - Secure payment handling
  - Custom data retention
  - Export capabilities
  - Self-hosted control
