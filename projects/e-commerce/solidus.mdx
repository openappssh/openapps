Solidus is a complete open source e-commerce solution built with Ruby on Rails. Originally forked from Spree, it focuses on providing a stable, extensible platform for building custom e-commerce experiences.

## Key Features

- **Modular Architecture**:

  - Component-based design
  - Service object pattern
  - Dependency injection
  - Event-driven system
  - Clean separation of concerns
  - Easy to extend

- **Complete E-commerce**:

  - Flexible product catalog
  - Advanced pricing rules
  - Order management
  - Multi-store support
  - Inventory management
  - Tax calculations

- **Developer-Focused**:

  - Well-documented API
  - Comprehensive test suite
  - Extension framework
  - Clear upgrade paths
  - Active community
  - Regular releases

- **Business Ready**:
  - Payment integrations
  - Shipping calculators
  - Promotion system
  - Customer accounts
  - Admin interface
  - Reporting tools
