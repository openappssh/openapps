Saleor is a modular, high-performance e-commerce platform built for the modern web. With its GraphQL-first approach and headless architecture, it enables developers to create unique shopping experiences across any channel.

## Key Features

- **GraphQL-First API**:

  - Complete GraphQL schema
  - Real-time subscriptions
  - Batch operations
  - Type-safe queries
  - API versioning
  - Webhook system

- **Modern Commerce**:

  - Multi-channel selling
  - Multi-warehouse inventory
  - Complex product variants
  - Dynamic pricing rules
  - Global tax support
  - Flexible discounts

- **Developer Experience**:

  - TypeScript SDK
  - React components
  - Extensive documentation
  - GraphQL playground
  - CLI tools
  - Example storefronts

- **Enterprise Ready**:
  - High performance
  - Horizontal scaling
  - Async task processing
  - Advanced caching
  - Multi-tenant capable
  - Plugin architecture
