Sylius is a modern e-commerce platform built on Symfony that provides maximum flexibility for complex e-commerce projects. It combines the power of Symfony components with e-commerce best practices.

## Key Features

- **Symfony Foundation**:

  - Built on Symfony 5+
  - Uses best components
  - Service-oriented
  - Event-driven architecture
  - Dependency injection
  - Professional patterns

- **Advanced Commerce**:

  - Complex product modeling
  - Flexible pricing
  - Multi-channel support
  - Advanced promotions
  - Tax engine
  - Shipping calculator

- **Developer-Friendly**:

  - BDD/TDD included
  - API Platform integration
  - State machines
  - Resource-based
  - Plugin architecture
  - Extensive docs

- **Enterprise Features**:
  - B2B capabilities
  - Multi-vendor ready
  - Workflow automation
  - High scalability
  - Performance optimized
  - Professional support
