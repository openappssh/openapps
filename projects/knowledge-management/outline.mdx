Outline is a modern team knowledge base and wiki designed to help growing teams organize and share their knowledge. It emphasizes collaboration, ease of use, and a beautiful reading experience while maintaining powerful features for technical teams.

## Key Features

- **Document Management**:

  - Real-time collaborative editing
  - Rich Markdown support
  - Version control and history
  - Document templates
  - Nested document structure
  - File attachments

- **Team Collaboration**:

  - Shared workspaces
  - Granular access controls
  - User groups
  - Comments and discussions
  - Activity tracking
  - Team notifications

- **Organization Tools**:

  - Collections and subfolders
  - Custom navigation
  - Powerful search
  - Tags and metadata
  - Cross-document linking
  - Table of contents

- **Integration & Security**:
  - Slack integration
  - REST API
  - SSO support
  - Audit logging
  - Data export
  - Self-hosting option
