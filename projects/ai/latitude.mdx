Latitude is an open-source platform for AI prompt engineering, deployment, and evaluation. It provides teams with tools to collaboratively design, test, deploy and maintain AI applications powered by large language models.

## Key Features

- **Prompt Management**:

  - Advanced prompt editor with PromptL support
  - Version control and collaboration features
  - Variables and conditional logic support
  - Template management
  - Collaborative editing capabilities

- **Development Tools**:

  - Interactive playground for testing
  - Batch testing capabilities
  - API endpoint deployment
  - Performance monitoring
  - Automatic interaction logging
  - SDK and API integrations

- **Evaluation System**:

  - LLM-based automated evaluation
  - Custom evaluation rules
  - Human review workflows
  - Test dataset management
  - Regression testing support
  - Performance metrics tracking

- **Deployment & Integration**:
  - API gateway for prompt deployment
  - SDK support for multiple languages
  - Seamless stack integration
  - Monitoring and observability
  - Access control and security
