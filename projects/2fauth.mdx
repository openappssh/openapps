2FAuth is a self-hosted two-factor authentication (2FA) manager that provides a secure way to store and access your 2FA tokens through a web interface. It offers an alternative to proprietary authenticator apps while giving you complete control over your authentication data.

## Key Features

### Token Management

- Store TOTP and HOTP tokens
- Import tokens from QR codes
- Bulk import/export capabilities
- Group and categorize accounts
- Search and filter functionality
- Password protection for sensitive tokens

### Security Features

- Encrypted storage of tokens
- Optional password protection
- Session management
- Secure API access
- Backup and restore functionality
- Activity logging

### User Experience

- Clean, modern web interface
- Mobile-friendly design
- Dark mode support
- Quick copy functionality
- Customizable settings
- Multiple language support

### Integration Options

- REST API for automation
- Browser extensions
- QR code scanner
- Import from other authenticator apps
- Export functionality for backups
- WebAuthn/FIDO2 support

2FAuth provides a robust solution for managing two-factor authentication tokens while maintaining complete control over your security data. Its self-hosted nature ensures your authentication secrets remain private and secure within your own infrastructure.
