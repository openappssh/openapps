SimpleX Chat is a revolutionary messaging platform that prioritizes privacy above all else. Unlike other messengers, it doesn't use phone numbers, usernames, or any other identifiers, making it the first messaging platform without user identities.

## Key Features

- **Unparalleled Privacy**:

  - No user identifiers of any kind
  - No phone numbers or emails required
  - No user IDs or usernames
  - Complete metadata protection
  - Anonymous connection establishment
  - Protects social graph

- **Advanced Security**:

  - Double ratchet end-to-end encryption
  - Post-quantum resistant encryption
  - Perfect forward secrecy
  - Decentralized architecture
  - No central servers or directories
  - Open-source and auditable

- **Full-Featured Messaging**:

  - Text and voice messages
  - File and media sharing
  - Group conversations
  - Disappearing messages
  - Message editing and deletion
  - Read receipts (optional)

- **Multi-Platform Support**:
  - Native iOS application
  - Native Android application
  - Desktop applications
  - Terminal-based client
  - Self-hostable relay servers
  - Decentralized network option
