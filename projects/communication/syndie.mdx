Syndie is a distributed forum system designed for anonymous and secure communication. It operates without central servers, using encrypted message syndication to create censorship-resistant discussion forums.

## Key Features

- **Distributed Architecture**:

  - No central servers required
  - Peer-to-peer synchronization
  - Offline-first design
  - Resilient to censorship
  - Archive import/export
  - Selective content sync

- **Privacy & Anonymity**:

  - Anonymous posting support
  - Pseudonymous identities
  - End-to-end encryption
  - No IP tracking
  - Secure channels
  - Private forums

- **Forum Functionality**:

  - Threaded discussions
  - Multiple forum support
  - File attachments
  - Message syndication
  - Moderation capabilities
  - Rich text formatting

- **Security Features**:
  - Message encryption
  - Digital signatures
  - Access control lists
  - Secure authentication
  - Protected channels
  - Tamper detection
