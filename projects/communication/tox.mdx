Tox is a peer-to-peer instant messaging and video calling protocol that offers end-to-end encryption. With no central servers, it provides truly private communication through a distributed network.

## Key Features

- **Decentralized Architecture**:

  - No central servers
  - Peer-to-peer connections
  - DHT-based user discovery
  - Resilient to censorship
  - No single point of failure
  - Community-run network

- **Complete Privacy**:

  - End-to-end encryption by default
  - No phone numbers required
  - No email registration
  - Anonymous Tox IDs
  - No metadata collection
  - No user tracking

- **Full Communication Suite**:

  - Instant text messaging
  - High-quality voice calls
  - Video calling support
  - File transfer capabilities
  - Group chat functionality
  - Screen sharing

- **Cross-Platform Support**:
  - Native desktop clients
  - Mobile applications
  - Multiple client choices
  - Consistent experience
  - Open protocol
  - Active development
