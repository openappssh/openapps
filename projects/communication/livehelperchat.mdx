Live Helper Chat is a comprehensive open-source live support solution that enables businesses to provide real-time customer support through their websites. It offers a rich set of features including chatbots, voice/video calls, and integrations with various messaging platforms.

## Key Features

- **Chat Capabilities**:

  - Real-time messaging
  - Voice and video calls
  - Screen sharing
  - File uploads
  - Typing indicators
  - Chat history
  - Multi-chat support

- **Support Tools**:

  - Canned responses
  - Visitor tracking
  - Chat transfers
  - Department routing
  - Performance analytics
  - Custom forms
  - Proactive chat

- **Integration Options**:

  - Mobile applications
  - Desktop clients
  - REST API access
  - Webhook support
  - ChatGPT integration
  - Social media platforms
  - SMS/WhatsApp

- **Technical Features**:
  - Docker support
  - Multi-language
  - Custom extensions
  - Template system
  - Cache support
  - API documentation
  - HTTPS support
