OwnTracks Recorder is the server component of the OwnTracks ecosystem, designed to store and manage location data from OwnTracks mobile applications. It provides a completely self-hosted solution for personal location tracking with an emphasis on privacy and data ownership.

## Key Features

- **Privacy-Focused Tracking**:

  - Complete data ownership
  - Self-hosted infrastructure
  - No third-party data sharing
  - Encrypted data transmission
  - Local data storage
  - Privacy by design

- **Location Data Management**:

  - Real-time location updates
  - Historical location storage
  - Geofence support
  - Location cards and annotations
  - Data export capabilities
  - Timeline visualization

- **Mobile Integration**:

  - iOS and Android apps
  - Battery-optimized tracking
  - Offline data queuing
  - Configurable accuracy
  - Background operation
  - Push notifications

- **Technical Features**:
  - MQTT protocol support
  - HTTP API access
  - Webhook integration
  - Lua scripting support
  - JSON data format
  - Web-based interface
