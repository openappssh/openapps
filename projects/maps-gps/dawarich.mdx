Dawarich is a self-hosted location history visualization and analysis tool that gives you complete control over your location data. It's designed as a privacy-focused alternative to Google Timeline, allowing you to visualize your movements, track your travel patterns, and analyze your location history without sharing data with third parties.

## Key Features

- **Location History Visualization**:

  - Interactive timeline view
  - Map-based visualization
  - Route reconstruction
  - Travel pattern analysis
  - Location frequency mapping
  - Time-based filtering

- **Data Import & Management**:

  - Google Takeout import
  - GPX file support
  - JSON data import
  - Bulk data processing
  - Data validation
  - Export capabilities

- **Privacy & Control**:

  - Complete data ownership
  - Self-hosted solution
  - No external data sharing
  - Local data storage
  - Encrypted data handling
  - Privacy-first design

- **Analysis & Insights**:
  - Travel statistics
  - Distance calculations
  - Most visited places
  - Time spent analysis
  - Movement patterns
  - Custom date ranges
