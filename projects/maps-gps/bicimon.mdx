Bicimon is a lightweight bike speedometer Progressive Web App that provides real-time cycling metrics directly in your browser. No app store downloads or installations required - just open and ride.

## Key Features

- **Real-time Tracking**:

  - Current speed display
  - Live distance measurement
  - Route recording
  - GPS-based tracking
  - Time elapsed
  - Average speed calculation

- **Progressive Web App**:

  - Works offline
  - App-like experience
  - No installation required
  - Cross-platform compatibility
  - Mobile optimized
  - Push notifications

- **Cycling Focused**:

  - Bike-specific metrics
  - Speed units (km/h, mph)
  - Distance units (km, miles)
  - Cycling-friendly interface
  - Quick start functionality
  - Pause/resume tracking

- **Privacy & Performance**:
  - Client-side processing
  - No data uploaded
  - Local storage only
  - Battery efficient
  - Lightweight application
  - Fast loading
