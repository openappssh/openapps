Maza is a simple yet effective local ad blocker that modifies your system's hosts file to block ads, trackers, and malicious domains. Unlike network-based solutions, it works directly on your device without requiring additional hardware or network configuration.

## Key Features

- **Local Ad Blocking**:

  - System-wide ad blocking
  - No network configuration needed
  - Works with all applications
  - No performance overhead
  - Instant activation
  - Offline functionality

- **Simple Operation**:

  - Single command installation
  - Easy start/stop control
  - Automatic blocklist updates
  - No complex configuration
  - Minimal dependencies
  - Lightweight footprint

- **Comprehensive Protection**:

  - Blocks ads and trackers
  - Malware domain blocking
  - Multiple blocklist sources
  - Custom domain support
  - Regular list updates
  - Community-maintained lists

- **Cross-Platform Support**:
  - Native Linux support
  - macOS compatibility
  - BSD systems support
  - Windows WSL compatible
  - Shell script based
  - POSIX compliant
