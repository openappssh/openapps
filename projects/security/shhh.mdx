Shhh is a lightweight secret sharing application designed to keep passwords and sensitive information out of emails and chat logs. It provides secure, temporary links with passphrase protection and automatic expiration.

## Key Features

- **Secure Secret Sharing**:

  - Passphrase-protected secrets
  - Configurable expiration times
  - One-time read functionality
  - Encrypted storage
  - Secure link generation
  - Automatic cleanup

- **Privacy-Focused**:

  - No user registration required
  - No activity logging
  - Self-destructing secrets
  - Anonymous usage
  - No tracking or analytics
  - Complete data control

- **Simple & Intuitive**:

  - Clean, minimal interface
  - Mobile-responsive design
  - Easy secret creation
  - Quick link sharing
  - Read receipt notifications
  - No complex setup

- **Easy Deployment**:
  - Docker container support
  - Python Flask framework
  - Minimal dependencies
  - Environment-based configuration
  - Lightweight resource usage
  - Production-ready
