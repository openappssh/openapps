QloApps is an open source hotel management and reservation system designed to help hotels take their business online. It provides a complete solution including property management, booking engine, and website builder through a modern web interface.

## Key Features

- **Property Management**:

  - Room inventory management
  - Multiple room types and rates
  - Booking calendar
  - Staff management
  - Guest profiles
  - Check-in/out management

- **Booking Engine**:

  - Online reservations
  - Real-time availability
  - Payment processing
  - Booking modifications
  - Email notifications
  - Multi-language support

- **Website Features**:

  - Customizable templates
  - Room showcase
  - Photo galleries
  - Content management
  - SEO optimization
  - Mobile responsive

- **Technical Features**:
  - REST API access
  - Multi-currency support
  - Reporting and analytics
  - Plugin architecture
  - Database-driven
  - Docker deployment option

## Who Should Use QloApps

QloApps is ideal for:

- Independent hotels
- Hotel chains
- Bed & breakfasts
- Guest houses
- Hostels
- Property managers

## Installation Requirements

- Apache/Nginx web server
- PHP 5.6+ to 7.4
- MySQL 5.1+ to 5.7
- 128MB PHP memory limit
- Required PHP extensions
- SSL certificate (for payments)
