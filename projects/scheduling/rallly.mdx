Rallly is an open-source scheduling platform built with Next.js that helps groups find the best meeting times by collecting participant availability. It provides a streamlined interface for creating polls and organizing events without endless email chains.

## Key Features

- **Meeting Coordination**:

  - Create availability polls
  - Participant voting system
  - Best time suggestions
  - Time zone handling
  - Group scheduling
  - Event management

- **User Experience**:

  - Clean, modern interface
  - Mobile-friendly design
  - No account required for voting
  - Quick poll creation
  - Real-time updates
  - Email notifications

- **Administration**:

  - User management
  - Admin control panel
  - Usage analytics
  - System monitoring
  - Custom branding options
  - Data export

- **Technical Features**:
  - Next.js framework
  - TypeScript codebase
  - PostgreSQL database
  - tRPC API
  - i18n support
  - Docker deployment

## Who Should Use Rallly

Rallly is ideal for:

- Teams coordinating meetings
- Event organizers
- Group activity planners
- Remote teams
- Anyone scheduling group events
- Organizations needing scheduling tools

## Installation Requirements

- Node.js 20+
- PostgreSQL database
- Docker (optional)
- Modern web browser
- 512MB RAM minimum
