Hi.Events is an open source event management and ticketing platform designed to help organizers sell tickets online for events of all sizes. It provides comprehensive tools for creating, managing and monetizing events through a modern web interface.

## Key Features

- **Ticketing & Sales**:

  - Multiple ticket types (free, paid, donation)
  - Capacity management
  - Promo codes and discounts
  - Product sales and add-ons
  - Custom taxes and fees
  - Partial/full refunds

- **Event Management**:

  - Real-time analytics dashboard
  - Custom event pages
  - Embeddable ticket widgets
  - SEO optimization tools
  - Offline event support
  - Multi-language interface

- **Attendee Features**:

  - Custom registration forms
  - QR code check-in
  - Digital ticket delivery
  - Email notifications
  - Bulk messaging
  - Data exports

- **Technical Features**:
  - REST API access
  - Webhook integrations
  - Role-based access control
  - Mobile-friendly interface
  - Stripe Connect integration
  - Advanced reporting

## Who Should Use Hi.Events

Hi.Events is ideal for:

- Conference organizers
- Workshop coordinators
- Concert promoters
- Festival managers
- Club event organizers
- Community event planners

## Installation Requirements

- Docker and Docker Compose
- Modern web browser
- 1GB RAM minimum
- Storage for event data
