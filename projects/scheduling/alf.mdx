Alf.io is an open source event attendance management system designed for event organizers who prioritize privacy, security and fair pricing for their customers. It provides comprehensive ticketing and registration features through a modern web interface.

## Key Features

- **Event Management**:

  - Create and manage multiple events
  - Flexible ticket categories and pricing
  - Custom registration forms
  - Waitlist management
  - Check-in system
  - QR code tickets

- **Payment Processing**:

  - Multiple payment gateway support
  - Secure payment handling
  - Refund management
  - Invoice generation
  - Promo code support
  - Multi-currency support

- **Attendee Features**:

  - Online ticket purchase
  - Digital ticket delivery
  - Apple/Google Wallet passes
  - Email notifications
  - Registration modification
  - Ticket transfer

- **Technical Features**:
  - REST API access
  - Multi-language support
  - Customizable templates
  - Database-driven
  - Mobile-friendly interface
  - Extensible architecture

## Who Should Use Alf.io

Alf.io is ideal for:

- Conference and event organizers
- Workshop coordinators
- Meetup organizers
- Trade show managers
- Organizations needing ticketing systems
- Privacy-focused event platforms

## Installation Requirements

- Java 17 or later
- PostgreSQL 10+
- Modern web browser
- 1GB RAM minimum
- Storage for event data
