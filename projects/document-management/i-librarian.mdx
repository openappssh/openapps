I, Librarian is an open-source document management system specifically designed for researchers and academics. It provides comprehensive tools for managing research papers, PDFs, and academic documents with features like metadata extraction and integration with major scientific databases.

## Key Features

- **Document Management**:

  - PDF file management and viewing
  - OCR capabilities with Tesseract
  - Office document import via LibreOffice
  - Metadata extraction and management
  - Full-text search functionality

- **Research Integration**:

  - Direct integration with PubMed
  - NASA database access
  - IEEE paper management
  - arXiv integration
  - DOI support and lookup

- **Collaboration Tools**:

  - Multi-user support
  - Group management
  - Shared library features
  - Access control settings
  - Remote access capabilities

- **Technical Features**:
  - Web-based interface
  - Apache/PHP architecture
  - Cross-platform support
  - Customizable deployment
  - Security controls

## Who Should Use I, Librarian

I, Librarian is ideal for:

- **Academic Researchers** managing research papers
- **Research Teams** needing collaborative document management
- **Libraries** organizing academic resources
- **Students** collecting and organizing research materials

## Getting Started

I, Librarian can be deployed through automated installers for Windows and Linux (DEB package), or manually installed on any platform supporting Apache and PHP. The system requires:

- Apache 2.4+
- PHP 7.2+
- Optional: LibreOffice for Office file import
- Optional: Tesseract for OCR support
