OpenSign is an open-source document e-signing solution designed to provide a secure, reliable and free alternative to commercial e-signature platforms. It focuses on making document signing accessible and straightforward for everyone while maintaining enterprise-grade security.

## Key Features

- **Document Signing**:

  - Secure PDF e-signing with encryption
  - Multiple signature types support
  - Document template creation
  - Expiring documents
  - Document rejection capability
  - Multi-signer workflow

- **Security & Verification**:

  - Email OTP verification
  - Detailed audit trails
  - Completion certificates
  - IP address tracking
  - Time-stamped activities
  - Encrypted storage

- **Document Management**:

  - OpenSign Drive
  - Template library
  - Folder organization
  - Document sharing
  - Status tracking
  - Batch processing

- **Integration & Customization**:
  - REST API access
  - Zapier integration
  - Cloud storage connectors
  - Custom email templates
  - CRM integrations
  - Enterprise platform support
