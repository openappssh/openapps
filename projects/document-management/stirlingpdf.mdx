Stirling PDF is a comprehensive, locally-hosted web application for PDF
manipulation. Built with security in mind, it processes files exclusively on
your infrastructure and provides a modern interface for all your PDF needs.

## Key Features

- **Extensive PDF Operations**: Over 50 different operations including
  splitting, merging, converting, rotating, compressing and more. The platform
  handles everything from basic PDF tasks to advanced document processing.

- **Security-First Design**: All files exist either client-side only or
  temporarily in server memory during processing. Downloaded files are
  immediately deleted from the server, ensuring your documents remain private.

- **Modern Interface**: Features a clean, intuitive web interface with dark mode
  support, parallel file processing, and custom download options. The GUI
  provides interactive tools for merging, splitting, rotating, and moving PDF
  pages.

- **Advanced Processing**: Supports OCR text recognition, PDF/A conversion,
  metadata editing, watermarking, digital signatures, and automated redaction.
  Convert between various formats including images, Word, PowerPoint, HTML, and
  more.

- **Enterprise Ready**: Offers optional authentication, SSO integration, custom
  pipelines for automation, API access, and database backup/restore
  capabilities. Available in 40+ languages with extensive localization support.
