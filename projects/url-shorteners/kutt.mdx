Kutt is a modern, feature-rich URL shortener designed for self-hosting. It provides extensive customization options, comprehensive analytics, and enterprise-grade features while remaining open source and free to use.

## Key Features

- **URL Management**:

  - Custom domain support
  - Password protection
  - Link expiration
  - Custom URLs
  - Visit tracking
  - API access

- **Analytics & Admin**:

  - Detailed visit statistics
  - Link management
  - User management
  - Private statistics
  - Admin dashboard
  - Real-time data

- **Integration Options**:

  - RESTful API
  - Browser extensions
  - Multiple database support
  - Redis caching
  - Docker deployment
  - API documentation

- **Deployment Features**:
  - Docker container support
  - Multiple database options
  - Easy configuration
  - Theme customization
  - Documentation
  - Active maintenance
