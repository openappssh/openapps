YOURLS is a lightweight, efficient URL shortener that you can install on your own server. It's one of the most established self-hosted URL shortening solutions, offering essential features while remaining simple to deploy and maintain.

## Key Features

- **URL Management**:

  - Custom short URL creation
  - Bulk URL importing
  - Private link options
  - Link expiration settings
  - Visit tracking
  - API access control

- **Analytics & Statistics**:

  - Detailed click tracking
  - Referrer monitoring
  - Geographic data
  - Visit history logs
  - Traffic source analysis
  - Real-time statistics

- **Integration Options**:

  - RESTful API
  - Plugin architecture
  - Browser bookmarklets
  - Multiple user support
  - Theme customization
  - API documentation

- **Deployment Features**:
  - Simple PHP requirements
  - MySQL database
  - Easy configuration
  - Plugin system
  - Comprehensive docs
  - Active community
