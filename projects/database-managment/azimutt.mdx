Azimutt is a modern database exploration tool that combines ERD visualization with powerful documentation and analysis capabilities. It's designed to handle real-world databases with hundreds of tables while maintaining usability through smart features like partial display and relation tracking.

## Key Features

- **Schema Exploration**:

  - Interactive ERD visualization
  - Smart layout management
  - Relation path finding
  - Partial schema display
  - Advanced search capabilities

- **Documentation Tools**:

  - Table and column notes
  - Custom tags and labels
  - Layout management
  - Team collaboration
  - Use case documentation

- **Data Analysis**:

  - Schema analysis
  - Best practices checking
  - Inconsistency detection
  - Performance insights
  - Database optimization

- **Integration Features**:
  - SQL file import
  - Direct database connections
  - Docker deployment
  - Kubernetes support
  - API access
