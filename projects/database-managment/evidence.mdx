Evidence is a business intelligence platform that takes a "BI as code" approach, allowing developers to create interactive data visualizations and reports using SQL queries embedded in markdown files. It bridges the gap between data analysis and web development.

## Key Features

- **Development Workflow**:

  - SQL queries embedded in markdown
  - Interactive chart components
  - Template-based page generation
  - Version control friendly
  - Hot reload development

- **Visualization Capabilities**:

  - Built-in chart components
  - Custom component support
  - Interactive data exploration
  - Responsive layouts
  - Dynamic filtering

- **Deployment Options**:

  - Evidence Cloud hosting
  - Self-hosted deployment
  - Static site generation
  - Netlify/Vercel support
  - Docker deployment

- **Integration Features**:
  - Multiple database support
  - SQL query integration
  - Version control systems
  - CI/CD pipelines
  - Authentication systems
