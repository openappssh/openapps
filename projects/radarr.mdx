Radarr is a fork of Sonarr adapted for automatically downloading movies via Usenet and BitTorrent. It can monitor multiple RSS feeds for new movies and will grab, sort, and rename them. It can also be configured to automatically upgrade the quality of existing files in the library when a better quality format becomes available.

## Key Features

### Movie Management

- Automatic movie searching, downloading, and processing
- Support for multiple download clients
- Quality-based profile system
- Automatic quality upgrades
- Movie list import from various sources
- Custom format support

### Integration Capabilities

- Integration with Plex, Emby, and other media servers
- Support for multiple indexers and download clients
- Blackhole download support
- Manual and automated search
- API for third-party integrations

### Organization Features

- Automated movie renaming and sorting
- Movie quality management
- Customizable naming schemes
- Library import and management
- Movie information and metadata from TMDb

### System Features

- Clean, modern web interface
- Mobile-first responsive design
- Detailed activity and history logging
- Backup and restore functionality
- Configurable update system
