What to Cook is an open-source recipe suggestion system built with Spring Boot and React that helps users decide what to cook based on ingredients they have available, eliminating the uncertainty of meal planning.

## Key Features

- **Recipe Suggestions**:

  - Ingredient-based recipe matching
  - Smart recipe recommendations
  - Available ingredients filtering
  - Recipe management interface
  - Simple, focused workflow
  - Mobile-friendly design

- **Technical Features**:

  - Spring Boot backend
  - React frontend
  - Docker deployment
  - Multi-architecture support (amd64/arm64)
  - Tailwind CSS styling
  - TypeScript codebase

- **User Experience**:
  - Clean, minimalist interface
  - Quick recipe suggestions
  - Easy ingredient selection
  - Responsive design
  - Session management
  - Intuitive navigation

## Who Should Use What to Cook

What to Cook is ideal for:

- Anyone struggling with "what to cook today?"
- Users wanting to cook with available ingredients
- Those seeking quick recipe suggestions
- People wanting to reduce food waste
- Anyone needing simple meal planning

## Installation Requirements

- Docker and Docker Compose
- Modern web browser
- 512MB RAM minimum
