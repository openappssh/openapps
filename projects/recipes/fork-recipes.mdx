Fork Recipes is an open-source recipe management system built with Django that focuses on simplicity and usability. It provides comprehensive features for managing food recipes, meal planning, and shopping lists, with additional AI-powered capabilities through OpenAI integration.

## Key Features

- **Recipe Management**:

  - Video or image-based recipes
  - Favorite recipe collections
  - Category organization
  - Clean print layouts
  - Recipe translations
  - Database backup/restore
  - Recipe scraping from URLs

- **Meal Planning**:

  - Built-in scheduling system
  - Shopping list generation
  - Ingredient management
  - Meal calendar view

- **AI-Powered Features**:

  - Recipe generation from ingredients
  - Multi-language translation support
  - Audio recipe narration (English)
  - Smart emoji integration
  - Intelligent recipe scraping

- **Technical Features**:
  - Django 5.x backend
  - Docker deployment
  - PostgreSQL support
  - SSL configuration options
  - API integration
  - Database snapshots

## Who Should Use Fork Recipes

Fork Recipes is ideal for:

- Home cooks wanting to organize their recipe collection
- Users looking for a self-hosted recipe management solution
- Those wanting AI-enhanced recipe features
- People needing meal planning and shopping list capabilities

## Installation Requirements

- Docker and Docker Compose
- OpenAI API token (for AI features)
- PostgreSQL or SQLite database
- SSL certificates (optional)
