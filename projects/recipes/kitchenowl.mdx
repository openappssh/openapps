KitchenOwl is an open-source grocery list and recipe management system built with Flutter and Flask. It provides comprehensive features for managing shopping lists, recipes, meal planning and household expenses with real-time synchronization across multiple users and devices.

## Key Features

- **Shopping List Management**:

  - Real-time sync across devices
  - Multi-user collaboration
  - Partial offline support
  - Quick item adding
  - Shopping history
  - Smart suggestions

- **Recipe Management**:

  - Recipe collections
  - Recipe sharing
  - Web recipe import
  - Ingredient scaling
  - Recipe photos
  - Cooking instructions

- **Meal Planning**:

  - Calendar view
  - Shopping list generation
  - Meal scheduling
  - Recipe suggestions
  - Plan sharing

- **Expense Tracking**:
  - Household balance tracking
  - Expense categories
  - Cost sharing
  - Transaction history
  - Balance reports

## Who Should Use KitchenOwl

KitchenOwl is ideal for:

- Households wanting to manage groceries collaboratively
- Home cooks organizing recipes and meal plans
- Users looking for a self-hosted solution
- Groups wanting to track shared expenses
- Anyone needing offline-capable shopping lists

## Installation Requirements

- Docker and Docker Compose
- PostgreSQL or SQLite database
- 1GB RAM minimum
- 500MB storage space
