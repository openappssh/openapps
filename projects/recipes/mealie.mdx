Mealie is an open-source recipe management system built with Python and Vue that focuses on providing a comprehensive solution for recipe organization, meal planning and shopping list management with an emphasis on family use and automation.

## Key Features

- **Recipe Management**:

  - Automated recipe imports from URLs
  - Recipe editor with rich text formatting
  - Recipe scaling and unit conversion
  - Recipe categorization and tagging
  - Multiple image support
  - Recipe sharing and printing
  - Nutrition information

- **Meal Planning**:

  - Weekly meal planner
  - Recipe scheduling
  - Shopping list generation
  - Calendar integration
  - Multi-user planning
  - Plan sharing

- **Shopping Lists**:

  - Automatic list generation from meal plans
  - Store section organization
  - Real-time sync
  - Multi-user collaboration
  - List history
  - Custom tags

- **Additional Features**:
  - REST API for integrations
  - Multi-language support (35+ languages)
  - User management and permissions
  - Data backup/restore
  - Docker deployment
  - Mobile-friendly interface

## Who Should Use Mealie

Mealie is ideal for:

- Families wanting to organize recipes collaboratively
- Home cooks needing meal planning capabilities
- Users wanting automated recipe imports
- Developers needing recipe API access
- Anyone seeking a comprehensive recipe management solution

## Installation Requirements

- Docker and Docker Compose
- 1GB RAM minimum
- Modern web browser
- Internet connection for imports
