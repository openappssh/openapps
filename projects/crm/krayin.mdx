Krayin CRM is an open-source customer relationship management solution built on Laravel and Vue.js frameworks. It provides a comprehensive platform for managing customer lifecycles, with features tailored for both SMEs and enterprises.

## Key Features

- **Core CRM Capabilities**:

  - Lead management and tracking
  - Customer lifecycle management
  - Custom attribute support
  - Role-based access control
  - Descriptive admin dashboard

- **Communication Tools**:

  - WhatsApp integration
  - VoIP integration with trunk calls
  - Email parsing via Sendgrid
  - Inbound route management
  - Multi-channel support

- **Development Framework**:

  - Laravel-based backend
  - Vue.js frontend
  - Modular architecture
  - REST API support
  - Extensive documentation

- **Customization Options**:
  - Custom attributes
  - Modular approach
  - UI/UX customization
  - Role management
  - Database flexibility
