Twenty is a modern open-source CRM platform that aims to provide a fresh alternative to traditional CRM systems. It focuses on delivering an intuitive user experience while maintaining complete data ownership and extensibility through its open-source nature.

## Key Features

- **Customer Management**:

  - Company and contact tracking
  - Opportunity management
  - Rich timeline views
  - Task management
  - Custom data fields

- **User Experience**:

  - Modern, intuitive interface
  - Keyboard shortcuts
  - Global search functionality
  - Customizable views
  - Rich text notes

- **Integration Capabilities**:

  - Email integration
  - REST API access
  - Webhook support
  - Future plugin system
  - Data import/export

- **Technical Foundation**:
  - TypeScript full-stack
  - React frontend
  - NestJS backend
  - PostgreSQL database
  - Redis caching
