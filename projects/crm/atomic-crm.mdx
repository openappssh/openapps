Atomic CRM is a modern, open-source customer relationship management system built with React and react-admin on top of Supabase. It provides a comprehensive set of features for managing contacts, tasks, deals and communications while maintaining full customization capabilities.

## Key Features

- **Contact & Task Management**:

  - Centralized contact organization
  - Task creation and reminders
  - Note-taking capabilities
  - Email capture and integration
  - Activity history tracking

- **Sales Pipeline**:

  - Kanban board visualization
  - Deal tracking
  - Sales pipeline management
  - Custom fields support
  - Data import/export tools

- **Integration & Access**:

  - Multiple authentication providers
  - REST API access
  - Email communication capture
  - Activity logging
  - Custom integrations

- **Customization Options**:
  - Custom fields
  - Theme customization
  - Component replacement
  - TypeScript/React extensibility
  - Docker deployment support
