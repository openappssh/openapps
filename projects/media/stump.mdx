Stump is a blazing-fast media server for comics, manga, and digital books. Built with Rust for exceptional performance, it provides a modern web interface and comprehensive API while maintaining low resource usage.

## Key Features

- **Performance First**:

  - Lightning-fast library scanning
  - Minimal memory footprint
  - Efficient thumbnail generation
  - Smart caching system
  - Rust-powered backend
  - Optimized for large libraries

- **Modern Interface**:

  - Clean, responsive web UI
  - Intuitive navigation
  - Grid and list views
  - Dark mode support
  - Mobile-optimized design
  - Real-time updates

- **Comprehensive Features**:

  - OPDS catalog support
  - Advanced metadata handling
  - Series organization
  - Smart collections
  - Tag-based filtering
  - Full-text search

- **Developer Friendly**:
  - RESTful API
  - GraphQL endpoint
  - Webhook support
  - Docker deployment
  - Active development
  - Extensive documentation
