Komga is a free and open-source media server specifically designed for comics, mangas, and graphic novels. It provides a beautiful web interface, powerful API, and seamless integration with popular comic reader apps.

## Key Features

- **Advanced Library Management**:

  - Multiple library support
  - Automatic metadata extraction
  - Series and collection organization
  - Custom reading lists
  - Duplicate detection
  - Bulk editing tools

- **Superior Reading Experience**:

  - Modern web-based reader
  - Webtoon/vertical scroll mode
  - Double-page spread support
  - Reading progress sync
  - Keyboard navigation
  - Touch gesture support

- **Extensive Integration**:

  - Full REST API
  - OPDS catalog feed
  - Tachiyomi extension
  - Chunky reader support
  - Server-sent events
  - Webhook notifications

- **Multi-User System**:
  - User account management
  - Per-library permissions
  - Age restriction controls
  - Shared reading lists
  - Activity dashboard
  - OAuth2 authentication
