BookLogr is a modern, self-hosted book library management system designed to help book lovers organize and track their reading. It provides an elegant interface for managing your personal library without relying on external services.

## Key Features

- **Book Management**:

  - Easy book cataloging
  - ISBN barcode scanning
  - Automatic metadata fetching
  - Cover image management
  - Multiple edition support
  - Series tracking

- **Reading Tracking**:

  - Track reading progress
  - Set and monitor reading goals
  - Reading statistics dashboard
  - Historical reading data
  - Personal notes and reviews
  - Rating system

- **Organization Tools**:

  - Custom shelf creation
  - Flexible tagging system
  - Advanced search filters
  - Multiple sort options
  - Bulk editing capabilities
  - Collection management

- **User Experience**:
  - Clean, modern interface
  - Mobile-responsive design
  - Data import/export
  - Privacy-focused
  - No external dependencies
  - Regular updates
