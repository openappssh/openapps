kiwix-serve is a lightweight HTTP server designed to serve offline content from ZIM files. It enables you to host complete copies of Wikipedia, Wiktionary, and other educational resources on your local network without internet access.

## Key Features

- **Offline Content Hosting**:

  - Serve complete Wikipedia offline
  - Host multiple content libraries
  - Support for all ZIM archives
  - Multi-language content
  - Regular content updates
  - Custom ZIM file support

- **User-Friendly Access**:

  - Clean web interface
  - Full-text search capability
  - Random article feature
  - Bookmark functionality
  - Mobile-responsive design
  - No app installation required

- **Technical Excellence**:

  - Lightweight C++ server
  - Minimal resource usage
  - No database required
  - Cross-platform support
  - Command-line interface
  - Docker deployment option

- **Content Variety**:
  - Wikipedia in all languages
  - Wiktionary dictionaries
  - Project Gutenberg books
  - TED Talks collection
  - Stack Overflow archives
  - OpenStreetMap data
