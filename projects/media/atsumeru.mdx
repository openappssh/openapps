Atsumeru is a comprehensive media server designed specifically for manga, comics, and light novels. It provides a complete ecosystem with native clients across all major platforms, making it easy to manage and read your digital library anywhere.

## Key Features

- **Comprehensive Library Management**:

  - Automatic metadata fetching
  - Series and volume organization
  - Category and tag support
  - Bulk import capabilities
  - Archive format support (CBZ, CBR, ZIP, RAR)
  - Smart collections

- **Multi-Platform Clients**:

  - Native Windows application
  - Native Linux application
  - Native macOS application
  - Android mobile app
  - Web-based reader
  - Consistent experience across platforms

- **Advanced Reading Features**:

  - Synchronized reading progress
  - Bookmark management
  - Multiple reading modes
  - Customizable reader settings
  - Offline reading support
  - Chapter navigation

- **Server Capabilities**:
  - Multi-user support
  - Role-based access control
  - RESTful API
  - OPDS catalog support
  - Automated library scanning
  - Metadata management
