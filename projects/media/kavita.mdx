Kavita is a lightning-fast, self-hosted digital library that supports a vast array of file formats. Built with performance in mind, it provides a sleek interface for managing and reading manga, comics, and books.

## Key Features

- **Extensive Format Support**:

  - Manga formats (CBZ, CBR, CB7, CBT)
  - Comic formats with metadata
  - eBook formats (EPUB, PDF)
  - Raw image folders
  - Archive file support
  - Automatic format detection

- **Superior Reading Experience**:

  - Fast, responsive web reader
  - Continuous vertical scrolling
  - Double-page spread support
  - Reading progress tracking
  - Bookmarks and annotations
  - Customizable reader settings

- **Smart Library Management**:

  - Automatic metadata fetching
  - Series and volume grouping
  - Smart collections
  - Advanced filtering
  - Rating and review system
  - Reading statistics

- **Multi-User Features**:
  - User account management
  - Age-based restrictions
  - Library sharing controls
  - Individual progress tracking
  - OPDS catalog support
  - Mobile app compatibility
