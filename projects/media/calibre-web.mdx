Calibre-Web is a web application that provides a clean, modern interface for browsing, reading, and downloading eBooks from an existing Calibre database. It offers remote access to your library with support for OPDS-compatible e-readers.

## Key Features

- **Web-Based Reading**:

  - Built-in eBook reader
  - EPUB, PDF, and CBZ/CBR support
  - Responsive reading interface
  - Reading progress sync
  - Customizable reader settings
  - Offline reading capability

- **Library Management**:

  - Browse existing Calibre database
  - Advanced search and filtering
  - Metadata viewing and editing
  - Book format conversion
  - Series and author browsing
  - Custom column support

- **E-Reader Integration**:

  - OPDS catalog server
  - Send books to Kindle
  - Download in multiple formats
  - E-reader app compatibility
  - Mobile-friendly interface
  - Kobo sync support

- **Multi-User Features**:
  - User account management
  - Permission-based access
  - Personal bookshelves
  - Reading history tracking
  - OAuth authentication
  - LDAP integration
